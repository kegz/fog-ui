# Shared UI

Reusable React UI components.

## Install
npm i @your-scope/shared-ui

## Usage
import { Button } from "@your-scope/shared-ui";

<Button variant="primary">Save</Button>

## Tailwind
Components use Tailwind utility classes.
Add this package to your Tailwind content paths:

./node_modules/@your-scope/shared-ui/**/*.{js,ts,jsx,tsx}
